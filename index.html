FreeStyle

Community LoRA Mining for style-reference stylization and dual-reference generation.

Style- and content-dual-reference generation aims to synthesize an image that preserves the structure and semantics of a content reference while adopting the visual style of a separate style reference. Despite recent progress in reference-based generation, this setting remains challenging because models must balance content fidelity, style alignment, and instruction following without leaking semantic content from the style reference. A key bottleneck is the lack of large-scale triplet data with clean content-style separation and broad long-tail style coverage. Existing data construction pipelines struggle to meet this need, as they either cover only a narrow set of styles, require costly manual curation, or produce triplets with imperfect content-style separation. In this work, we propose FreeStyle, a scalable dual-reference generation framework based on community LoRA mining. We treat community LoRAs as compositional anchors for style and content, and design a rigorous generation and filtering pipeline to construct large-scale Style-Reference (SRef) and Content-Reference (CRef) triplets across multiple base models. We also introduce a benchmark covering style-reference and content-style dual-reference generation, with evaluations on style similarity, content preservation, aesthetics, instruction following, and leakage rejection. To improve content-style disentanglement beyond data construction, FreeStyle further introduces an attention-level constraint that suppresses style-reference leakage while preserving style richness, along with a RoPE weighting method, leading to a better balance among style alignment, content fidelity, and leakage control.

Dataset

A dataset split into SRef and CRef + SRef evaluation

The benchmark covers style-only stylization and the harder dual-reference setting. Both sections below use the same triplet grammar so the relationship between references, prompt, and target remains legible.

SRef dataset

Style Transfer Pair Generated By Style Trigger

In the style transfer dataset pair, we collect effective style trigger words from the community, using Nano Banana model to transfer the style of the collected content image including many different types of objects, like people, animals, and scenes.

CRef + SRef dataset

Content and Style Reference Pair Generated By Community LoRA

We mine the community LoRAs to find the effective style and content LoRAs, and then combine them to generate the dual-reference dataset. The dual-reference dataset covers a wide range of styles and content, which is more challenging for the model to learn the disentanglement of style and content.
